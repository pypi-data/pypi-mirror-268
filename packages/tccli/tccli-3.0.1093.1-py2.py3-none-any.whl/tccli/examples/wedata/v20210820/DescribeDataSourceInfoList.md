**Example 1: 获取数据源信息**

获取数据源列表

Input: 

```
tccli wedata DescribeDataSourceInfoList --cli-unfold-argument  \
    --PageNumber 0 \
    --PageSize 10 \
    --ProjectId 1 \
    --Type MYSQL
```
